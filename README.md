# Treap Data Structure
A treap is a data structure which combines binary tree and binary heap (hence the name: tree + heap  
=> Treap).

### Time Complexity in Big O Notation:

* Best Case, Average Case, Worst Case
* Search: log n
* Insert: log n
* Delete: log n
* Space O(n)
